# Reviews

Use this directory for internal and external reviews of benchmark design,
annotation quality, baseline validity, and paper positioning.

Current review priority:

1. Is the benchmark sufficiently different from missing-modality robustness?
2. Are recoverability and evidence labels reliable enough to publish?
3. Does the baseline suite test modality triage rather than only final-task
   accuracy?
4. Are dataset licenses and generated corruptions documented clearly enough for
   an AAAI benchmark submission?
